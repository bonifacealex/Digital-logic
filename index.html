About Number Systems

Decimal (Base 10)

Uses digits 0-9. The most common number system used in daily life.

Binary (Base 2)

Uses digits 0 and 1. Fundamental to computer systems and digital electronics.

Octal (Base 8)

Uses digits 0-7. Commonly used in computing and digital systems.

Hexadecimal (Base 16)

Uses digits 0-9 and A-F. Widely used in programming and computer science.

Roman Numerals

Uses letters I, V, X, L, C, D, M. Ancient numeral system still used today.
